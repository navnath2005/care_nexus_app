🏥 CareNexus

CareNexus is a Flutter + Firebase healthcare management application that connects patients, doctors, pharmacies, ambulances, and administrators on a single secure platform.
It focuses on emergency response, role-based access, and real-time healthcare coordination.

🚀 Key Features
🔐 Firebase Authentication (Email, Phone OTP, Google)
👥 Role-based access (Patient, Doctor, Pharmacy, Ambulance, Admin)
🚑 Emergency ambulance request & tracking
🩺 Medical profile & vital monitoring (IoT-ready)
🛡️ Secure Firestore rules
📊 Admin dashboard & user management

🧱 Tech Stack
Flutter – Cross-platform UI
Firebase Auth – Secure authentication
Cloud Firestore – Real-time database
IoT (MAX30102) – Heart rate & SpO₂ monitoring
Google Maps API – Location tracking
